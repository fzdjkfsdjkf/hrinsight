# Краткая инструкция

## Как запустить

1) Установите зависимости:

`pip install streamlit pandas numpy plotly openpyxl`

2) Запустите приложение:

`streamlit run hriinsight.py`

Откроется веб-интерфейс: загрузите свой CSV или нажмите «Использовать демо-данные 2022–2025».

## Как подготовить свой CSV

1) Файл должен быть в формате CSV (UTF-8). Рекомендуемые колонки:

2) employee_id — уникальный идентификатор сотрудника

3) full_name — Ф.И.О.

4) hire_date — дата приёма (формат YYYY-MM-DD или любой, распознаваемый библиотекой pandas)

5) termination_date — дата увольнения (или пусто, если сотрудник в компании)

6) department — подразделение

7) position — должность

8) gender — M/Ж или прочее

9) age_at_hire — возраст при приёме (число)

10) performance_score — оценка (1–5)

## Пример строки:

1001,Иванов Иван,2022-05-12,2023-07-01,Продажи,Менеджер по продажам,М,29,3.8
