# Краткая инструкция

## Как запустить

Установите зависимости (если ещё не установлены):

pip install streamlit pandas numpy plotly openpyxl

Сохраните код в файл hriinsight.py.

Запустите приложение:

streamlit run hriinsight.py

Откроется веб-интерфейс: загрузите свой CSV или нажмите «Использовать демо-данные 2022–2025».

Как подготовить свой CSV

Файл должен быть в формате CSV (UTF-8). Рекомендуемые колонки:

employee_id — уникальный идентификатор сотрудника

full_name — Ф.И.О.

hire_date — дата приёма (формат YYYY-MM-DD или любой, распознаваемый pandas)

termination_date — дата увольнения (или пусто, если сотрудник в компании)

department — подразделение

position — должность

gender — M/Ж или прочее

age_at_hire — возраст при приёме (число)

performance_score — оценка (1–5)

Пример строки:

1001,Иванов Иван,2022-05-12,2023-07-01,Продажи,Менеджер по продажам,М,29,3.8
